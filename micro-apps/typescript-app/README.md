# TypeScript 全面学习项目

这个项目旨在涵盖所有 TypeScript 的核心知识点和使用方法，通过实际代码示例和详细注释帮助你全面掌握 TypeScript。

## 项目结构

```
typescript-learning/
├── src/                    # 源代码目录
│   ├── 01-basic-types/     # 基础类型
│   ├── 02-interfaces/      # 接口
│   ├── 03-classes/         # 类
│   ├── 04-functions/       # 函数
│   ├── 05-generics/        # 泛型
│   ├── 06-advanced-types/  # 高级类型
│   ├── 07-modules/         # 模块
│   ├── 08-namespaces/      # 命名空间
│   ├── 09-decorators/      # 装饰器
│   ├── 10-utility-types/   # 实用工具类型
│   ├── 11-type-inference/  # 类型推断
│   ├── 12-type-guards/     # 类型守卫
│   ├── 13-jsx-tsx/         # JSX/TSX
│   └── app/                # 示例应用
│       └── index.ts        # 入口文件
└── dist/                   # 编译输出目录
```

## 安装与运行

1. 安装依赖：
```bash
npm install
```

2. 运行项目：
```bash
npm start
```

3. 开发模式（自动重启）：
```bash
npm run dev
```

4. 构建项目：
```bash
npm run build
```

## 学习路径

1. 从基础类型开始，了解 TypeScript 的类型系统
2. 学习接口和类，掌握面向对象编程
3. 深入函数和泛型的使用
4. 探索高级类型和类型操作
5. 学习模块化和命名空间
6. 掌握装饰器等高级特性
7. 通过示例应用整合所有知识点

每个模块都包含详细的代码注释，帮助你理解每个概念和用法。
